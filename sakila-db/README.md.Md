# 🎬 Sakila SQL Project

A beginner-friendly SQL project using the **Sakila sample database** to practice real-world analysis with MySQL and phpMyAdmin.  
Includes 10 professional queries exploring customers, films, staff, and revenue insights.

- SQL Joins & Grouping  
- Aggregation (SUM, COUNT, AVG)  
- CASE & Date Formatting  
- Data analysis for business insights
